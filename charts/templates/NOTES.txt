== Backup CronJob ==
  Schedule: {{ .Values.cron.schedule }}
  The CronJob "{{ include "duplicacy.fullname" . }}-cron" will run backups on the configured schedule.
  View recent jobs:
    kubectl get jobs -l app.kubernetes.io/component=cron -n {{ .Release.Namespace }}
  View logs of the latest job:
    kubectl logs -l job-name=$(kubectl get jobs -l app.kubernetes.io/component=cron -n {{ .Release.Namespace }} --sort-by=.metadata.creationTimestamp -o jsonpath='{.items[-1].metadata.name}') -n {{ .Release.Namespace }}
{{- end }}

== Prometheus Exporter ==
  Metrics are exposed on port {{ .Values.exporter.port }}.
  Access the metrics endpoint:
    kubectl port-forward svc/{{ include "duplicacy.fullname" . }}-exporter {{ .Values.exporter.port }}:{{ .Values.exporter.port }} -n {{ .Release.Namespace }}
    curl http://localhost:{{ .Values.exporter.port }}/metrics
  Health check:
    curl http://localhost:{{ .Values.exporter.port }}/health
